# Restro

Aplikasi pengurusan / pesanan restoran berasaskan Laravel 10 dan PHP 8.1+, dengan pilihan **aplikasi desktop (Electron)**.

**Sokongan Windows (desktop):** **Windows 7 ke atas** (melalui Electron 22.x; lihat nota dalam dokumentasi penuh).
